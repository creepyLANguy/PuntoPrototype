Frequently Asked Questions

Do players need phones to score?

No. Pulse and Beacon devices work independently on court, so players can focus on the game.

Can scoreboards be viewed remotely?

Yes. Matches can be viewed in real-time from any internet-connected display, including phones and TVs.

Do devices work on battery power?

Yes. All devices include rechargeable batteries with USB-C support for continuous power if needed.

Can clubs start with only a few devices?

Absolutely. Padel Push is designed to be modular and scalable to fit any club size.
